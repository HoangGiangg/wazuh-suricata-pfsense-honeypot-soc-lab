\# Lab Scope



\## Scope



This project focuses on building and documenting a defensive cybersecurity lab for SOC monitoring and incident analysis.



The lab simulates a small enterprise-style network that includes a firewall, IDS/IPS, SIEM, honeypot services, and protected internal servers.



\## In Scope



\- pfSense firewall configuration

\- Suricata IDS/IPS deployment

\- Wazuh SIEM deployment

\- Honeypot deployment using Cowrie, Heralding, and Snare/Tanner

\- Server-real hardening and monitoring

\- Mail and FTP service monitoring

\- Log collection from Honeypot, Suricata, and server-real

\- Custom detection rules

\- Automated redirect and IP blocking

\- Email alerting

\- AI-based alert risk classification

\- Incident report documentation

\- MITRE ATT\&CK mapping



\## Out of Scope



\- Attacking real public systems

\- Collecting real victim data

\- Uploading raw sensitive logs

\- Uploading VM disk files

\- Uploading private keys, passwords, tokens, or credentials

\- Production deployment



\## Safety Notes



All attacks and simulations were performed inside an isolated lab environment.
